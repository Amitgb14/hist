Welcome to Bash history across multiple machine's documentation!
================================================================


How to run
-----------

::

    $ python main.py

::

    $ python main.py --sync [sync all machine]

::

    $ python main.py --m<id>



Add new node
-------------

::

    $ssh-copy-id <node> 


Write Host file
---------------

host.cfg

#id   node

1     127.0.0.1
